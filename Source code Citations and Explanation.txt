These are the following features I've added to my Roll-a-Ball game : 

Background music / collectible pickup noises
Basic enemy/enemies for waypoints
Respawn points
Win and lose states
Map assets (nature packs for world building)
Jump Action
Pause/resume/main menu button
Seamless transitioning between scenes and levels
AI Enemy
